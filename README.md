# Simple Calculator

A basic calculator application built with HTML, CSS, and JavaScript that performs arithmetic operations and allows file reading.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (ร)
- Division (รท)
- Error handling for division by zero
- File reading capability (text files)

## How to Use

1. Open `index.html` in a web browser
2. Enter two numbers in the input fields
3. Click on the desired operation button to calculate the result
4. The result will be displayed in the output area

## File Reading

To read a file:
1. Select a text file using the file input
2. Click the "Read File" button
3. The file content will be displayed in the output area

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Requirements

Any modern web browser that supports HTML5 and JavaScript.
